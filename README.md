## Yuefou
使用 GraphQL 的 DEMO 项目

## Back-end stack
* database: mysql 5.7.1
* database api proxy layer: prisma
* graphql server: nodejs + graphql-yoga
* static server: nginx 1.10.3(ubuntu)
* track: apollo-engine

## Front-end stack
* mvvm framework: angular 5
* store management: apollo-client

## DevOps
* pm2
